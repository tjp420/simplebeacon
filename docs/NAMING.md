# Simplebeacon — Naming & Branding

**Product name:** **Simplebeacon**  
**Website:** [simplebeacon.ai](https://simplebeacon.ai)  
**npm package:** `simplebeacon`  
**CLI command:** `simplebeacon` (alias: `samplebeacon` for one release)  
**Config directory:** `.simplebeacon/` (legacy `.samplebeacon/` still read if present)

## Rename map (Samplebeacon → Simplebeacon, May 2026)

| Before | After |
|--------|-------|
| Samplebeacon | **Simplebeacon** |
| `samplebeacon` npm / CLI | `simplebeacon` |
| `.samplebeacon/` | `.simplebeacon/` |
| `packages/samplebeacon-cli/` | `packages/simplebeacon-cli/` |
| `web/samplebeacon-dashboard/` | `web/simplebeacon-dashboard/` |
| `npm run samplebeacon:*` | `npm run simplebeacon:*` (root npm legacy aliases removed May 2026; CLI bin `samplebeacon` still works) |
| Report type `samplebeacon-report` | `simplebeacon-report` |

The earlier **Samplebeacon** name was an internal typo; public branding and domain were always **Simplebeacon**.
